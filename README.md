This mod adds a variety of nuclear physics features, from beryllium decay to helium to a fusion reactor that can explode and completely destroy your factory!
Created as an experiment to test the capabilities of neural networks, this mod was generated entirely by neural networks. The code and all logic were generated by the DeepSeek neural network, and the textures were generated using Nano Banana 2 and Seedream 5.0.
